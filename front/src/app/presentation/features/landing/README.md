# Landing Feature

## Purpose
Public entry experience for FinTreX visitors before authentication.

## Routes
| Path | Component | Description |
|---|---|---|
| `/` | `LandingHomePage` | Main landing page with hero and top navigation |
| `/products` | `Products` | Product capabilities overview |
| `/pricing` | `Pricing` | Plan and package overview |
| `/about` | `About` | Company introduction |
| `/trial` | `Trial` | Limited-time demo/trial entry page |

## Notes
- This feature stays outside the authenticated app layout.
- Theme/language controls are public UI controls shown in the landing top bar.
